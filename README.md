# Super Mauro Bros


## Sobre
Trabalho realizado para a matéria de Inteligência Artificial na graduação de Engenharia da Computação na UNICEP São Carlos.


## Como iniciar
Instale as dependências do projeto:

```
pip install pygame
```

Execute o arquivo main.py
